## Overview

AV Commons is a platform enabling members to contribute to collaborative projects while prioritizing spiritual growth. The system remains under development, using itself as a prototype.

## Notification Settings

**Default Setting:** No notifications (opt-in required)

### Available Options

| Code | Setting | Description |
|------|---------|-------------|
| `p` | Personal | Use your personal default setting for group notifications |
| `n` | New | Immediate alerts when resources become relevant |
| `d` | Daily | Single daily notification digest |
| `w` | Weekly | Single weekly notification digest |
| `x` | None | No notifications (check dashboard manually) |

## Providing Feedback

### Three Methods

1. **Comments**: Highlight text and select the comment icon
   - Mobile: Enable edit mode, highlight text, access menu, choose comment

2. **Suggestions**: Highlight text and activate suggestion mode via icon
   - Mobile: Access top menu to toggle "Suggest changes"

3. **Alternate Versions**: Create a document copy for substantial modifications
   - Link it to the original document
   - Submit through AVC Functions

### Additional Channels

- Feedback form for direct input
- Error reports via AV Commons Error Report Form
- Group discussions in group spaces on AV website
- Group Dashboard as central information hub
